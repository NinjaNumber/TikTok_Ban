# TikTok_Ban
Sentiment Analysis using VADER to examine public opinions toward Tiktok ban proposal.

This study aims to recognize how people feel about the government’s approach to suspend TikTok’s usage, in an attempt to mitigate national security threats and regulate censored political speech. It would be interesting to see whether TikTok has truly gained liking and loyalty from its user base.
